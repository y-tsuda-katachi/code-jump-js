# code-jump-js

## はじめに

これは、コードジャンプのjQueryを含む練習サイトをネイティブなJavaScriptに変換しデモサイトとしてまとめたリポジトリです。

JavaScriptの基本構文、DOM操作などを一通り学んだ後、
練習サイトの模写を通じてJavaScriptの理解を深めることを目的としています。

## 使い方

下記のデモサイトリンクからアクセスし、デモを確認してください。

その後、各自の環境で模写コーディングを行います。

> 模写コーディングのやり方については
> 本家・[Codejump](https://code-jump.com/)を参照し確認してください
